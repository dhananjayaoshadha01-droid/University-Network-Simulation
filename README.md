## 🚀 Project Overview
The simulation represents a modern campus network with specialized zones for academic activities.

### 🏢 Network Sections:
* **Admin Office (VLAN 10):** Secure wired network for administrative tasks.
* **IT Lab (VLAN 20):** High-performance network for practical sessions.
* **Study Area / Library (VLAN 30):** Wireless Connectivity for students.
* **Lecture Hall (VLAN 40):** Multimedia-ready network for teaching.
